## 📝 INTRODUCTION

### 🐾 개발 동기
> 새해☀️를 맞이하여 올해에는 자신이 가지고 싶은 습관이 있을 것 입니다. </br> 지키기 어려운 작심삼일러분들을 위해 지속적인 체크로 습관을 가질 수 있도록 도움을 주고자 개발하게 되었습니다 🤗
### 🐾 개발 목표
> 사용자가 직접 습관을 입력하고 그것을 체크합니다.</br>
> 그 과정에서 자신의 꾸준함을 기록할 수 있게 하고 지난 날을 성찰할 수 있게 하는 것을 목표로 두었습니다.
